# Shared Examples

## Purpose

- store compact, reusable examples for skills and workflows;
- keep the examples portable across Claude, Codex/OpenAI, and Gemini wrappers.

## Recommended Example Families

- skill-level input/output samples;
- workflow handoff packets;
- preview and reporting summaries;
- platform-specific examples only when the wrapper adds real value.

## Included Portable Examples

- `session-packet.json` — core orchestrator memory packet;
- `forecast-summary.json` — compact forecast handoff;
- `qa-decision.json` — go / no-go packet after preview and QA;
- `deployment-summary.json` — deployment and monitoring follow-up packet.

## Rule

- keep examples business-reviewed, short, and aligned with the real tool names.
