# Available CLI Commands

## Creative Discovery

- `orbiads creatives list --json` `[free]` — list all creatives in the network.
- `orbiads creatives list --search "<name>" --json` `[free]` — search creatives by name.
- `orbiads creatives get --id <id> --json` `[free]` — get details for a specific creative.

## Creative Upload

- `orbiads creatives upload --file <path> --name "<name>" --advertiser <id> --json` `[5 credits]` — upload a new creative asset.
