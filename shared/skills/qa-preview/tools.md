# Allowed Tools

## Compliance and SSL Checks

- `scan_creative_compliance` `[variable]` — scan a creative payload or asset bundle for compliance findings.
- `validate_creative_ssl` `[free]` — validate SSL for one creative.
- `validate_creative_ssl_batch` `[variable]` — validate SSL for a batch of creatives.

## Preview and Coverage

- `get_preview_urls` `[variable]` — generate previews at line-item scope.
- `get_campaign_preview_urls` `[variable]` — generate previews at order scope.
- `check_creative_coverage` `[free]` — detect missing or inconsistent creative coverage before activation.
