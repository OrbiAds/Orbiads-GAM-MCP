# Available CLI Commands

## Targeting Discovery

- `orbiads inventory keys --json` `[free]` — list all custom targeting keys and their values.
- `orbiads inventory keys --key "<name>" --json` `[free]` — get values for a specific targeting key.

## Ad Unit Qualification

- `orbiads inventory ad-units --search "<name>" --json` `[free]` — search ad units to validate targeting compatibility.
- `orbiads inventory ad-units --id <id> --json` `[free]` — get full details including sizes and targeting for a specific ad unit.
