# Available CLI Commands

## Ad Unit Discovery

- `orbiads inventory ad-units --json` `[free]` — list all active ad units in the network.
- `orbiads inventory ad-units --search "<name>" --json` `[free]` — filter ad units by name.
- `orbiads inventory ad-units --id <id> --json` `[free]` — get details for a specific ad unit.

## Placement Discovery

- `orbiads inventory placements --json` `[free]` — list all placements in the network.
- `orbiads inventory placements --search "<name>" --json` `[free]` — filter placements by name.

## Targeting Keys

- `orbiads inventory keys --json` `[free]` — list custom targeting keys and their values.
- `orbiads inventory keys --key "<name>" --json` `[free]` — get values for a specific targeting key.
