# Available CLI Commands

## Campaign Inspection

- `orbiads campaigns get <id> --json` `[free]` — get full campaign configuration and status.

## Creative Validation

- `orbiads creatives get --id <id> --json` `[free]` — inspect a specific creative for compliance and SSL status.

## Dry-Run Deployment

- `orbiads campaigns deploy <id> --dry-run --json` `[free]` — simulate deployment without executing. Reports warnings and errors.
