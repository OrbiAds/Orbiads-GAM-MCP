# Allowed Tools

## Asset and Template Preparation

- `create_image_creative` `[free]` — upload or create the image creative asset when the Native payload needs it.
- `ensure_classic_native_template` `[free]` — ensure the Classic Native template exists before creative creation.

## Native Creative Writes

- `create_classic_native_creative` `[free]` — create the Native creative from the approved payload.
- `associate_creative` `[free]` — associate the created creative with the approved line items.
