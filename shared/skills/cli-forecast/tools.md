# Available CLI Commands

## Forecast

- `orbiads reporting run --type forecast --ad-units <ids> --start <date> --end <date> --json` `[free]` — run an availability forecast for the specified ad units and date range.

## Supporting Reads

- `orbiads inventory ad-units --json` `[free]` — list ad units to select forecast targets.
- `orbiads inventory ad-units --id <id> --json` `[free]` — get details for a specific ad unit.
