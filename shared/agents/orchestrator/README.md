# Orchestrator

## Role

- interpret the user goal;
- select the smallest useful shared skill or workflow;
- protect credit usage, confirmation boundaries, and session continuity.

## Inputs

- current user request;
- last confirmed tenant, network, advertiser, order, line item, and creative context;
- open blockers, approvals, and budget sensitivity.

## Outputs

- chosen skill or workflow;
- short execution plan;
- structured handoff packet for the delegated skill-agent.
