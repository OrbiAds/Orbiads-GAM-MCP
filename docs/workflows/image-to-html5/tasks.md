# Tasks

## Delivery Preparation

- confirm sizes, dates, and targeting before packaging the HTML5 bundle;
- resolve the destination advertiser, order, and line items;
- keep the final asset bundle version frozen before upload.

## Upload and QA

- upload the exact approved files through the confirmed HTML5 tool;
- associate the creative only after bundle review;
- run compliance, SSL, preview, and coverage checks before deployment.

## Handoff

- return the creative IDs, association result, preview packet, and unresolved issues;
- move next to the deploy gate in the `campaigns` skill for approval and activation, followed by delivery verification via `reporting`.
