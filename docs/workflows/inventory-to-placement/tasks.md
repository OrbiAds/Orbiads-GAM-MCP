# Tasks

## Scope and Read

- confirm the active tenant and network;
- collect the approved ad unit subset or generate it through the inventory skill;
- capture the inventory audit summary and unresolved anomalies.

## Placement and Targeting Alignment

- compare selected ad units against existing placements;
- identify whether placement create, update, or archive is required;
- read the existing custom targeting taxonomy before proposing new keys or values;
- confirm geo, language, device, and Native / Fluid constraints.

## Approval and Handoff

- validate naming, grouping, and destructive changes with humans;
- execute approved writes only after the review packet is accepted;
- hand off `selectedAdUnitIds`, placement IDs, targeting IDs, remaining risks, and any forecast summary.
